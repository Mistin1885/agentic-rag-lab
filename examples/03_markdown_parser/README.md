# 03 Markdown Parser

Goal: parse Markdown files into stable document, section, and chunk records.

This exercise is planned for the next step. It should produce records with:

- `doc_id`
- `file_path`
- `title`
- `frontmatter`
- `heading_path`
- `section_id`
- `line_start`
- `line_end`
- `text`

This prepares the repo for metadata-aware retrieval and Vespa indexing.
